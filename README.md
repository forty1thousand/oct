## Description
Schedule appointments with doctors online

## Features
- Accepting and declining appointments
- Paginated scroll when viewing doctors 
- Password are hashed server-side for security

## Tech stack
- Backend
    - Actix-Web
    - Diesel
    - Postgresql
    - Redis
- Frontend
    - Nextjs
    - Material ui
    - React Query
